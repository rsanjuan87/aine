Doble significado

Anti Inflamatorio No Esteroide
para el ecosistema Android

Los antiinflamatorios no esteroideos (AINEs) actúan directamente en el origen del dolor sin ser soluciones invasivas. No son esteroides — no transforman el sistema, solo reducen la fricción donde se necesita.


AINE funciona igual. No es un emulador — no traduce instrucciones de CPU. No es un contenedor completo — no arrastra un kernel Linux. Es una capa de traducción de APIs: los syscalls Linux se mapean a XNU, el Binder corre sobre Mach IPC, OpenGL ES se convierte en Metal. El código de la app corre directamente en Apple Silicon, como si siempre hubiera sido una app macOS.

Cómo se compara AINE

Emulador
BlueStacks / QEMU
Emula hardware ARM completo. CPU virtual, GPU virtual. Lento. Alto consumo. La app nunca toca el metal real.
Contenedor
Waydroid
Android completo en LXC + kernel Linux real. Solo funciona en Linux. No disponible en macOS.
AINE ✦
Traducción de API
El código corre nativo en ARM64. Solo se traducen las llamadas. Sin overhead de VM. Como Wine, pero para Android en macOS.
